# Security Questionnaire Desk

A tiny answer desk for startup security questionnaires.

## Buyer

B2B SaaS founders and tiny teams selling into larger companies before they are ready for a full GRC platform.

## Problem

Security questionnaires block deals and consume founder/engineer weekends through 100+ question spreadsheets and repetitive portal forms.

## MVP boundary

Questionnaire import → reusable answer library → confidence flags → export back to spreadsheet/doc. No SOC2 automation, legal advice, compliance certification, or invented claims.

## Launch offer

$99 for one redacted questionnaire pack: answer-library starter, first-draft response map, confidence flags, and follow-up questions for the few answers only the team can verify.

Lightweight product artifact: `answer-map-template.csv` demonstrates the response-map shape and the no-invention guardrail.

Target product pricing remains $19–$49/month for answer-library reuse after the manual pack proves the workflow.

## Evidence basis

- HN evidence: tiny teams describe 100+ question security reviews, 250-row Excel sheets, and weekend engineer time.
- Conveyor/Vanta-style platforms prove budget exists but are oversized for the pre-GRC moment.
- Sekorti-like tools prove active demand around SaaS founder questionnaire workflows.
